The openspace wilderness was created in the Fall/Winter of 2013/2014 for the 2014 Turbulence Net Art Exhibition

It was built with django 1.6 using python, jquery, html, and css. Every line of code aside from the django and jquery APIs was written from scratch. In addition to the actual build there are a number of utility function, adobe scripts, and other tidbits included with the git repository.
